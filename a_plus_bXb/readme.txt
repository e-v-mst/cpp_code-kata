Задача «a+b*b»
Имя входного файла: 	input.txt
Имя выходного файла: 	output.txt
Ограничение по времени:    2 секунды
Ограничение по памяти: 	256 мегабайт

В данной задаче требуется вычислить значение выражения a+b*b.

Формат входного файла
Входной файл состоит из одной строки, которая содержит два целых числа a и b. Для этих чисел выполняются условия -10^9 ? a ? 10^9, -10^9 ? b ? 10^9

Формат выходного файла
В выходной файл выведите единственное целое число — результат сложения a+b*b.

Пример 1
input.txt
23 11
output.txt
144

Пример 2
input.txt
-100 1
output.txt
-99
